# Ground Truth

> An AI detective game whose narrator cannot lie about the case. Built for the Agents League hackathon, Creative Apps track.

## Microsoft IQ

Uses **Foundry IQ** (agentic retrieval on Azure AI Search) to ground every turn in the case canon, so the narrator never invents evidence or contradicts the timeline.

## Tech

React + Vite, Vercel serverless functions, Azure AI Search (Foundry IQ), Anthropic Claude.

In active development.
